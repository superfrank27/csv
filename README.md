# 像素村庄保卫战（GitHub Pages 一键发布）

这是一个纯前端网页游戏（`index.html` + `style.css` + `game.js`）。

## 当前核心玩法
- **第三人称视角**（可看到自己角色与装备）。
- 默认武器刀剑，可买手枪/步枪，枪械弹药耗尽自动切回刀剑。
- 武器特效：刀剑挥舞、枪械子弹飞行。
- 僵尸为像素方块风格，并从地图四面的古墓持续刷新。
- 大地图包含山、树、蓝天、村庄（3栋房子+商店+农田+围墙）。
- 围墙每段可被僵尸攻击3次破坏；无僵尸靠近时村民会修墙。
- 村民会种地、收玉米、消耗玉米并繁衍；村民被咬会转化为僵尸。
- 村里有猪羊；可击杀回复1血；僵尸也会攻击动物并回血。

## 一键发布到 GitHub Pages

> 前提：仓库默认分支是 `main`。

1. 推送到 `main`。
2. 进入仓库 **Actions**。
3. 运行 **Deploy static game to GitHub Pages** 工作流。
4. 首次到 **Settings → Pages** 确认 Source 为 **GitHub Actions**。
5. 打开：`https://<你的用户名>.github.io/<仓库名>/`

## 本地运行

```bash
python3 -m http.server 4173
```

浏览器打开：`http://127.0.0.1:4173`
