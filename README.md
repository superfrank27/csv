# 像素村庄保卫战（GitHub Pages 一键发布）

这是一个纯前端网页游戏（`index.html` + `style.css` + `game.js`）。

## 当前核心玩法
- **第三人称视角**，可看见角色、武器、盔甲。
- 场景升级为更明显的像素风（蓝天、山体、树林、村庄、商店、农田、围墙、古墓）。
- 默认刀剑，商店可买手枪/步枪，枪械子弹打完自动切回刀剑。
- 武器特效：刀剑挥舞动作 + 枪械子弹飞行 + 基础音效。
- 僵尸从四面古墓刷新，攻击围墙 3 次可破坏，村民可在安全时修墙。
- 村民、僵尸、动物加入防穿透碰撞分离（玩家可跳跃越墙）。
- 村庄内有猪羊与玉米循环：击杀动物玩家回复 1 血，僵尸可吞食动物回血。

## 一键发布到 GitHub Pages

> 前提：仓库默认分支是 `main`。

1. 推送到 `main`。
2. 进入仓库 **Actions**。
3. 运行 **Deploy static game to GitHub Pages** 工作流。
4. 首次到 **Settings → Pages** 确认 Source 为 **GitHub Actions**。
5. 打开：`https://<你的用户名>.github.io/<仓库名>/`

## 本地运行

```bash
python3 -m http.server 4173
```

浏览器打开：`http://127.0.0.1:4173`
